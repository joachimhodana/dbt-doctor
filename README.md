# dbt Doctor

Your agent writes bad dbt. This catches it.

**Site:** [dbt-doctor.joachimhodana.com](https://dbt-doctor.joachimhodana.com)

```bash
npx dbt-doctor@latest
```

## Requirements

- Node.js 22+
- sqlfluff: `pip install sqlfluff sqlfluff-templater-dbt`
